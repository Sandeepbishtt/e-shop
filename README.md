# e-Shop

e-Shop is a shopping website made using React JS and Redux. 

## Tech Stack

**Client:** React, Redux, Semantic UI

**Server:** FakeAPI

  
## Run Locally

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Deployment

To deploy this project run

```bash
  npm run deploy
```
